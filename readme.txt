Este programa es una versióon actualizada y mejorada del juego base que se descargo en la libreria freegames para python

El programa base corre un juego llamado memory, donde el jugador debera voltear tarjetas hasta encontrar dos números iguales.
Si el jugador voltea dos tarjetas y no son iguales vuelven a su posicion escondida original, si son iguales se eliminan y descubren una parte de la imagen detrás. 
El juego termina cuando todas las tarjetas fueron retiradas y la imagen es completamente visible.

Nuestros cambios hacen que
-se agrege un contador de clics
-detecta cuando todos los cuadros han sido destapados
-centran los números en los cuadros

El proyecto para la Semana Tec actualmente esta terminado y no requiere de más actualizaciones

En caso de que se quiera probar este juego se requiere de la instalación previa de Python y la libreria de FreeGames. 
Posteriormente se debe correr en la terminal de la computadora
